# Utilities Module

This directory contains utility functions and common components for the WebOrdering_Automation system. These utilities provide shared functionality used across different modules of the system.

## Contents

- `__init__.py` - Python package initialization
- `logger.py` - Logging utilities and configuration
- `metrics.py` - Metrics computation and evaluation utilities
- `validators.py` - Input validation and data validation utilities
- `vietnamese_processor.py` - Vietnamese language processing utilities

## Overview

The utilities module provides common functionality that is used across different components of the WebOrdering_Automation system. These utilities help maintain consistency and reduce code duplication across the codebase.

## Components

- **Logger**: Provides structured logging functionality with configurable levels and formats
- **Metrics**: Implements computation of various evaluation metrics used throughout the system
- **Validators**: Contains validation functions for input data and configuration values
- **Vietnamese Processor**: Specialized utilities for processing Vietnamese text and language-specific features

## Architecture

The utilities are organized to be lightweight and reusable across all modules. They follow a functional approach where appropriate and provide clear interfaces for common operations needed throughout the system.
