# Paper Figures

This directory contains figures and visualizations used in the research paper describing the WebOrdering_Automation system. These figures illustrate key concepts, experimental results, and system architecture.

## Contents

- `ablation_study.pdf` - Figure showing ablation study results
- `annotation_quality.pdf` - Figure showing annotation quality metrics
- `architecture.pdf` - Figure showing system architecture
- `data_flow.pdf` - Figure showing data flow in the system
- `results_comparison.pdf` - Figure showing comparison of results
- `trajectory_example.pdf` - Figure showing example trajectories

## Overview

The figures directory contains visual representations that support the narrative of the research paper. These include architectural diagrams, experimental result visualizations, and examples that help readers understand the system and its performance.

## Figure Categories

- **Architecture diagrams**: Visualize the system structure and component relationships
- **Experimental results**: Show performance comparisons and ablation studies
- **Examples**: Provide concrete illustrations of system behavior
- **Data visualizations**: Show dataset characteristics and quality metrics

## Usage

These figures are referenced in the LaTeX paper source and are included in the final document. They are typically generated using scripts in the `scripts/paper/` directory.
