# Supplementary Materials

This directory contains supplementary materials referenced in the research paper describing the WebOrdering_Automation system. These materials provide additional information that supports the main paper but is too detailed for the main text.

## Contents

- `annotation_prompts.pdf` - Detailed annotation prompts used in the study
- `full_schema.pdf` - Complete schema documentation
- `quality_samples.pdf` - Sample of high-quality annotated data
- `sample_trajectories.pdf` - Examples of collected trajectories

## Overview

The supplementary materials provide additional details that support the main paper without cluttering the primary narrative. These materials include examples, extended documentation, and detailed information about the methodology.

## Material Categories

- **Annotation materials**: Detailed prompts and examples for the annotation process
- **Schema documentation**: Complete specifications for data schemas
- **Quality samples**: Examples demonstrating the quality of collected and annotated data
- **Trajectory examples**: Sample trajectories showing the system's behavior

## Usage

These materials are referenced in the main paper and provide additional context and details for interested readers. They are typically included as supplementary files in the paper submission.
