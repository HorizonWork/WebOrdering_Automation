# Paper

This directory contains materials related to the research paper describing the WebOrdering_Automation system. It includes figures, LaTeX source files, tables, and supplementary materials.

## Contents

- `figures/` - Images and diagrams used in the paper
- `latex/` - LaTeX source files for the main paper
- `supplementary/` - Supplementary materials and additional resources
- `tables/` - Table definitions and data for paper

## Overview

The paper directory is organized to support the academic publication process. The LaTeX files contain the main paper content, while figures and tables provide visual and numerical results. Supplementary materials contain additional information that supports the main findings.

## Structure

- The `latex/` directory contains the main paper source files in LaTeX format
- The `sections/` subdirectory within `latex/` contains individual paper sections
- The `figures/` directory contains visualizations and diagrams used in the paper
- The `tables/` directory contains LaTeX table definitions and data
- The `supplementary/` directory contains additional materials referenced in the paper

## Building the Paper

To compile the paper, navigate to the `latex/` directory and use a LaTeX compiler with support for the ACM template (acmart.cls).
